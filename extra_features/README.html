Grawlix Bonus Features

Grawlix 1.5 has the capability, with some extra files, to enable a media library and add the ability to host multiple comics within one installation.

Adding these new features isn’t as hard as it looks. Mostly you’ll upload files using FTP.

Adding Multibook

Grab the file book.list.php and drop it in your _admin folder. That's it.

(eishiya's note: Using this feature can lead to some name/ID conflicts that will break things. Use at your own risk!)

Adding the Media Gallery

Grab the following files and upload them to your _admin folder:
media.edit.php, media.list.php, media.thumbnails.php, and media.usage.php

That's it!

Once they're in, log into your admin panel and check out your brand new features!
